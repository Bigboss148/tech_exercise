## Getting Started


```bash
git clone https://github.com/KeisukeMiyamoto1324/tech_exercise.git
```

```bash
npm install
```

```bash
npm install -g json-server
```

```bash
json-server --watch db.json --port 5001
```

```bash
npm run dev
```
